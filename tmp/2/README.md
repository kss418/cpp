# 2번 연습 문제

| 파일 | 구현 내용 |
| --- | --- |
| `2-1.cpp` | `is_streamable`로 스트림 출력 가능 여부를 판정해 `print`에서 출력 가능 시 값 출력, 아니면 `N/A` 출력. |
| `2-2.cpp` | 산술 타입만 허용하는 `add`를 제공하고 그 외 타입은 삭제된 오버로드로 컴파일 에러 유도. |
| `2-3.cpp` | 이동 대입/이동 생성 가능 여부를 합쳐 `swap_like` 활성화를 제약하고 불가능 시 삭제. |
| `2-4.cpp` | `at_like`에서 `x.at(i)` 우선, 불가 시 `x[i]`를 선택하도록 requires 오버로드 분기. |
| `2-5.cpp` | `a == b` 결과를 `bool`로 변환 가능한 경우에만 `equal_like`를 허용하고 그 외는 삭제. |
| `2-6.cpp` | `<` 비교 가능 시 그 결과로, 불가 시 `size()` 비교로 `min_like`를 선택하며 참조를 유지. |
| `2-7.cpp` | 안전한 수명을 보장하는 조건에서만 `read_view`로 `std::string_view` 반환하도록 우선순위 제약. |
| `2-8.cpp` | `emplace_back` → `push_back` → `insert(end, ...)` 순으로 `emplace_one`을 선택해 원소를 삽입. |
