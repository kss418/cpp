# 3번 연습 문제

| 파일 | 구현 내용 |
| --- | --- |
| `3-1.cpp` | `sum_all`로 전달된 인자를 모두 더해 합을 반환하도록 fold를 구현. |
| `3-2.cpp` | `all_of`/`any_of`로 각 인자에 predicate를 적용해 모두/하나라도 참인지 판정. |
| `3-3.cpp` | `print_csv`로 인자를 CSV 형식(`, ` 구분)으로 출력하고 개행. |
| `3-4.cpp` | `type_count<T>`로 인자 중 `T`와 같은( cv/ref 제거 ) 타입 개수를 집계. |
| `3-5.cpp` | `zip_for_each`로 두 튜플의 같은 인덱스 원소를 묶어 순서대로 호출. |
| `3-6.cpp` | `join_to_string`으로 구분자를 넣어 문자열을 합쳐 `std::string` 반환. |
| `3-7.cpp` | `transform_reduce`로 `transform` 후 `op`를 좌측부터 누적하는 mini 버전 구현. |
| `3-8.cpp` | `overload_set`으로 여러 callable을 합성한 `overloaded` 패턴 구현. |
